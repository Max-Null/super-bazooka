# 配置参考

本项目（cc-gui）为 Tauri 2 + Vue 3 桌面应用，无 Spring Boot 风格的 `@ConfigurationProperties` 配置类。配置项管理方式：

- **前端设置**：通过 SettingsPanel 配置 API Key / Base URL / Model，持久化到 `localStorage`（key: `cc-gui-settings`）
- **字号设置**：小/中/大（14/18/22px），通过 `data-font-size` 属性 + CSS `rem` 实现全局缩放
- **主题**：亮/暗，通过 `data-theme` 属性切换 CSS 变量
- **Ponytail 模式**：lite/full/ultra，持久化到 `localStorage`
