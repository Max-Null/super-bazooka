# 前端-会话

## 功能说明

会话侧边栏——会话列表、新建、删除、重命名。管理 Claude Code 会话生命周期。

- SessionSidebar：会话列表侧边栏，支持搜索过滤、新建会话、删除会话、重命名会话
- 所有操作通过 session store → tauri-bridge → Rust 后端执行

## 公开 API

| 类型 | 名称 | 说明 |
|------|------|------|
| component | SessionSidebar | 会话侧边栏组件 |

## 依赖说明

| 模块 | 说明 |
|------|------|
| `stores` | useSessionStore（会话 CRUD） |
| `lib` | Tauri 桥接层 |
