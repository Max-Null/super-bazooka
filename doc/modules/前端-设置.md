# 前端-设置

## 功能说明

设置面板——API 配置、连接测试、主题语言切换。配置 DeepSeek API 代理参数。

- SettingsPanel：设置面板，API Key/Base URL/Model 配置，DeepSeek 连接测试，主题（亮/暗）和语言（中/英）切换

## 公开 API

| 类型 | 名称 | 说明 |
|------|------|------|
| component | SettingsPanel | 设置面板组件 |

## 依赖说明

| 模块 | 说明 |
|------|------|
| `stores` | useSettingsStore（API/主题/语言持久化） |
| `lib` | Tauri 桥接层（connect_llm 连接测试） |
