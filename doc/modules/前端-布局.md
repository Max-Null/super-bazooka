# 前端-布局

## 功能说明

应用主布局——全局键盘快捷键、主题属性绑定。应用的顶层容器。

- AppShell：应用主布局容器，含全局键盘快捷键注册（Ctrl+K 命令面板等）、`data-theme` 属性绑定（亮/暗主题切换）、`data-font-size` 属性绑定（小/中/大字号）

## 公开 API

| 类型 | 名称 | 说明 |
|------|------|------|
| component | AppShell | 应用主布局容器 |

## 依赖说明

| 模块 | 说明 |
|------|------|
| `stores` | settings store（主题/语言/字号） |
| `composables` | useCommandPalette（全局快捷键） |
