# 前端-国际化

> vue-i18n 中英双语翻译文件。

## 功能说明

- 所有用户可见字符串的中英双语翻译
- `zh.json`：中文翻译文件
- `en.json`：英文翻译文件
- 支持参数化消息（`{error}` 占位符）

## 结构

```mermaid
graph LR
    App[Vue App] --> I18n[vue-i18n 实例]
    I18n --> ZH[zh.json 中文]
    I18n --> EN[en.json 英文]
    Settings[settings store] -->|locale: zh|en| I18n
    I18n --> Components["所有组件 $t('key')"]
```

## 公开 API

| 类型 | 名称 | 说明 | 文件 |
|------|------|------|------|
| i18n | zh.json | 中文翻译 locale 文件 | src/locales/zh.json |
| i18n | en.json | 英文翻译 locale 文件 | src/locales/en.json |

## 配置属性

本模块无对外配置属性。

## 代码示例

```typescript
// 组件中使用
import { useI18n } from "vue-i18n";
const { t } = useI18n();
const errorMsg = t("error.claudeNotFound"); // → "未找到 Claude Code CLI，请先安装"
```
