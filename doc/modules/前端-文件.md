# 前端-文件

## 功能说明

文件浏览面板——目录树、文件预览、Diff 对比。浏览项目文件和工作区内容。

- FilePanel：文件浏览主面板，包含 FileTree
- FileTree：目录树组件，右键菜单（打开/预览/在资源管理器显示）
- FilePreview：文件内容预览（代码/Markdown/图片/二进制检测）
- DiffViewer：行级差异对比视图

## 公开 API

| 类型 | 名称 | 说明 |
|------|------|------|
| component | FilePanel | 文件浏览面板 |
| component | FileTree | 目录树组件（右键菜单） |
| component | FilePreview | 文件内容预览 |
| component | DiffViewer | 差异对比视图 |
