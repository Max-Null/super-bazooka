# 变更记录

## [0.4.0] — 2026-07-04

### 消息时间线渲染

- CC 原始 content 块顺序保持——思考、工具调用、文本按时间线交替显示
- 时间线 UI：竖线连接圆点（💭思考/🔧工具/💬文本），按类型着色
- 思考块默认折叠，仅流式中最后一段展开，完成后显示内容预览
- 计时分离：总耗时（⏱）+ 纯思考时间（🧠）分开统计
- 工具卡片实时显示执行计时（⚡），长操作（>5s）珊瑚色高亮
- 状态行：思考中（波浪闪烁）→ 已完成 / 已停止

### 会话机制审计修复

- 工作区切换：先停止旧 CC 进程再创建新会话
- 会话删除：终止 CC 进程防止后台残留
- 会话切换竞态：3 处 activeSessionId guard 丢弃过期结果
- 跨模式 isProcessing：stream-event 和 process-exited 按 zenMode 区分活跃会话
- SessionCache LRU 淘汰：上限 20，命中移到末尾
- stop_session 锁优化：锁内提取→锁外等待→重新获取清理
- plan 模式审批 sendStdin 加 try-catch，防弹窗卡死
- 空 assistant 消息直接删除不显示残留气泡
- 最终消息去重：迟到 assistant 事件用已完成消息内容做 startsWith 比较

### MD→docx 简化

- 移除 pandoc 路径，统一走 CC /docx skill
- 检测改用 Rust 后端 `check_skill_installed`（扫 ~/.claude/skills/+ 插件缓存）
- 已安装直接发指令，未安装先让 CC 全局安装

### 工作区与文件

- 刷新后 FilePanel 从 settings.cwd 恢复路径
- 切换工作区/目录时清空文件预览

### 颜色系统

- CSS 变量统一管理，禁止硬编码色值
- 新增 --blue 变量，dot-* 系列改用 var(--accent)/var(--coral)/var(--blue)
- 非 Anthropic 后端隐藏 costUSD 显示

### 开发规范

- CLAUDE.md 新增 Git 工作流：积累不微提交、审查后提交、推送前 squash
- CLAUDE.md 新增颜色系统速查表

## [0.2.0] — 2026-07-02

### 禅模式（直接 LLM 对话）

- 🧘 Header 按钮切换禅模式——跳过 CC CLI，直接调 LLM chat/completions API（SSE 流式）
- 禅模式下隐藏工具栏（权限模式/Effort/Ponytail 对直接 LLM 无效）
- Zen 会话独立管理：`mode` 字段区分 CC/Zen，DB 持久化，刷新不丢
- 多轮对话上下文支持——自动加载历史消息构建完整 messages 数组

### 侧边栏重构（Rail）

- 左侧改为 Rail 窄栏（40px 常驻）+ 可展开面板（260px），收起时显示会话首字符按钮
- Rail 内按顺序：☰ 展开/收起 → ＋ 新建 → 会话历史（首字符，英文大写）
- 展开时 rail 隐藏，面板顶部 ☰ 收起按钮
- 汉堡按钮从 Header 移除

### 活动指示点

- 会话列表项显示状态点：🟢 绿闪烁（处理中）/ 🔵 蓝（未读）/ 🟠 橙闪烁（等待审批）
- Rail 角标 + 展开面板内联，同源 `sessionActivity` 数据

### 会话切换消息缓存

- 切走后台会话继续接收流式事件（`handleBackgroundStreamEvent` 写入 `sessionCache`）
- 切回时优先从缓存恢复（含切走期间产生的新消息），缓存未命中则从 DB 加载
- `process-exited` / `stream-event` 按 `session_id` 路由，不干扰活跃会话

### Provider Logo

- 禅模式按钮和指示器显示当前 Provider 官方 logo（lobe-icons CDN，与设置面板同源）
- `PROVIDER_LOGOS` 集中在 `settings.ts` 模块导出

### 代码质量

- 新增 `useSessionSwitch` composable——会话切换逻辑统一入口，消除 AppShell/SessionSidebar 重复
- `saveCurrentConfig` 加 500ms 防抖——避免每次按键触发 SQLite 写入
- 4 个 scroll watch 合并为 1 个
- `connectedMcpServers` 从模块级 ref 移入 session store
- `appVersion` 补充声明、`AttachedFile` 从 store 导入、冗余 `v-if` 删除

## [0.1.1] — 2026-06-27

### NDJSON 交互协议 (P0/P1)

- **P0-1**: CC 子进程启动参数修正 — 添加 `--input-format stream-json` 和 `--permission-prompt-tool stdio`，移除 `--print`，消息通过 stdin NDJSON 发送
- **P0-2**: 修正 `control_response` 格式为嵌套结构 `response.subtype/request_id/response.behavior/updatedInput/message`，新增 `request_id` 往返和 `set_permission_mode` 初始化
- **P1-1**: `hook_callback` 不再自动允许，转发前端走用户审批流程
- **P1-2**: `AskUserQuestion` 问答弹窗 — radio/checkbox 选项组 + Other 自由输入，答案格式 `{questions, answers}`
- **P1-3**: interrupt 优雅停止 — 先发 `control_request(subtype: "interrupt")`，3 秒超时兜底
- **P1-4**: deny 含 `message`、allow 含 `updatedInput`（含在 P0-2）

### i18n 修复

- 修复 vue-i18n v10 消息编译器 `SyntaxError: 10` — `@` 用 `{'@...'}` 转义，`{ }` 字面量修正
- 授权弹窗 "Allow Bash?" 国际化 — `chat.allowTool: "允许使用 {tool}？"`
- 工具名翻译 — `Bash→命令行`、`Write→写入文件`、`Read→读取文件` 等，中英双份

### UI 改进

- 思考计时重构 — 每段 tool_use 前独立计时（`🧠Xs`），总耗时 = 各段求和，去除实时计时器，不受审批暂停影响
- Debug 面板复制按钮 — 与消息气泡同款 SVG 图标，鼠标移入显示
- 问答弹窗样式修复 — header 标签 `whitespace-nowrap`、去掉双滚动条

### 工程

- 版本号管理 — 从 `package.json` 注入 `__APP_VERSION__` 到 UI，三处同步（`package.json` / `tauri.conf.json` / `Cargo.toml`）
