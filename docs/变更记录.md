## [0.10.0] — 2026-07-14

### Excel 选区修复与增强

- **拖拽选区修复**：表头行（`<th>`）选区 tip 不显示 → `queryCell()` 同时匹配 `td,th`
- **选区视觉高亮**：拖拽时 `.xlsx-selected` class 实时标记选中单元格

- **空白处清理**：点击 padding 区域自动清除旧选区并关闭 tip

### 选区 tip 统一

- **MD 编辑器选区接入**：废弃未渲染的 `mdTipPos`，CodeMirror 选区改用统一 `selTip`
- **建议输入框**：MD 编辑器 tip 含修改建议输入框，回车发送
- **滚动更新**：编辑器滚动时 tip 位置跟随更新
- **Sheet 切换清理**：切换 Excel Sheet 时自动清除选区状态

### 选区卡片合并

- `DomSnippet` + `TextSnippet` → 统一 `{ content, label }` 数据结构
- 统一 chip 样式（复用 `.attach-chip`），标签格式 `[E]` `[T]` `[M]` `[D]` + 文件名
- 发送时自动拼到消息前面，输入框写额外指令
- 右键菜单「添加到会话」修复：ChatPanel 监听 `attach-files` CustomEvent

### MD 预览增强

- **大纲面板**：☰ 切换，左侧 200px 标题导航，点击平滑滚动
- **标题补全**：h4/h5/h6 解析 + CSS 递减字号、标题自动生成 id
- **样式补全**：多行引用、任务列表、删除线、图片
- **链接**：`target="_blank"` + `shell.open` 在默认浏览器打开
