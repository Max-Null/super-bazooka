## 1. 快速上手

### 1.1 启动前配置

1. 点击右上角齿轮图标 → **设置**
2. 填写 API Key、Base URL、选择模型
3. 点击 **连接测试**，确认 API 可用
4. 切换语言：中文 / English
5. 切换主题：深色 / 浅色 / 跟随系统

### 1.2 发送第一条消息

1. 左侧 **新建会话**（或 Ctrl+N）
2. 底部输入框输入内容，按 **Enter** 发送
3. 助手回复实时流式显示，思考过程可展开查看

## 2. 聊天

### 2.1 消息发送

| 操作 | 方式 |
|------|------|
| 发送 | Enter |
| 换行 | Shift+Enter |
| 停止生成 | 点击输入框右侧红色停止按钮 |
| 重新发送 | 悬停用户消息 → 点击重发按钮 |
| 编辑后重发 | 悬停用户消息 → 点击编辑按钮 → 修改内容 → Enter |

### 2.2 附件

- **拖拽文件**到输入框区域即可附加
- 点击工具栏 **📎 按钮** 选择文件
- 附件以芯片形式显示在输入框上方，可逐个移除

### 2.3 模式与深度

工具栏左侧三个下拉菜单：

| 菜单 | 选项 | 说明 |
|------|------|------|
| **权限模式** | 编辑前询问 / 自动编辑 / 计划模式 / 自动模式 / 完全放行 / 无询问 | 控制工具调用的审批策略 |
| **思考深度** | low / medium / high / xhigh / max / ultracode | 控制推理深度，越深越慢但越细致 |
| **精简模式** | 关闭 / 轻量 / 标准 / 极致 | Ponytail 懒人模式强度，选中即执行 |

### 2.4 消息展示

- **Markdown 渲染**：标题、列表、代码块（语法高亮 14 种语言）、Mermaid 图表
- **思考过程**：默认折叠，点击展开查看推理链
- **工具调用**：显示调用的工具名和参数摘要，可展开查看完整 JSON
- **统计信息**：助手回复完成后显示耗时、token 消耗、费用
- **流式光标**：回复进行中显示闪烁光标，token 实时追加

### 2.5 权限审批

当 Claude 需要执行工具调用（读写文件、执行命令等）时，聊天区顶部出现审批栏：
- 显示工具名称
- **允许** / **拒绝** 按钮
- 可勾选 "始终允许此类操作"

### 2.6 调试日志

聊天区底部可折叠的调试面板，显示：
- 原始流事件
- stderr 输出
- 错误信息

便于排查 CLI 通信问题。

## 3. 会话管理

左侧边栏列出所有历史会话。

| 操作 | 方式 |
|------|------|
| 新建 | 侧边栏顶部 + 按钮，或 Ctrl+N |
| 切换 | 点击会话列表项 |
| 搜索 | 侧边栏顶部搜索框，实时过滤 |
| 重命名 | 悬停会话 → 点击编辑图标（或 F2） |
| 删除 | 悬停会话 → 点击垃圾桶图标（或 Del） |
| 导出 | 命令面板 → "导出当前会话为 Markdown" |
| 侧边栏开关 | Ctrl+B |

每条会话显示标题、token 总量、累计费用。

## 4. 文件浏览

右侧文件面板（Ctrl+E 打开/关闭）。

### 4.1 基本操作

- **目录树**：点击文件夹展开/折叠子目录
- **面包屑导航**：顶部路径分段，点击可跳转任意层级
- **回到根目录**：面包屑左侧 🏠 按钮

### 4.2 文件操作

| 操作 | 方式 |
|------|------|
| 预览 | 单击文件 → 面板下方内联预览 |
| 全屏预览 | 双击文件 → 弹窗 |
| 右键菜单 | 在资源管理器中打开 / 复制路径 / 复制文件名 |

### 4.3 文件预览

自动检测类型并适配展示：

| 类型 | 展示方式 |
|------|----------|
| 图片（PNG/JPG/GIF/WebP/SVG） | 大图模式 |
| 代码 | highlight.js 语法高亮 |
| Markdown | 格式化渲染 |
| 文本 | 纯文本 |
| 二进制 | 友好提示 |

### 4.4 Diff 对比

传入两段文本，行级差异显示：
- 🟢 绿色：新增行
- 🔴 红色：删除行
- 默认色：不变行

## 5. 设置

右上角齿轮图标进入。

### 5.1 API 配置

| 配置项 | 说明 |
|--------|------|
| API Key | DeepSeek 或兼容 OpenAI 的密钥 |
| Base URL | API 地址（默认 `https://api.deepseek.com`） |
| Model | 模型选择，后缀标注上下文窗口 |

点击 **连接测试** 验证配置。

### 5.2 外观

| 设置 | 选项 |
|------|------|
| 语言 | 中文 / English |
| 主题 | 深色 / 浅色 / 跟随系统 |
| 字号 | 小(14px) / 中(18px) / 大(22px) |

导航栏右侧太阳/月亮图标可一键切换主题。

### 5.3 行为

| 设置 | 选项 | 说明 |
|------|------|------|
| 权限模式 | 6 种 | 默认权限策略 |
| 思考深度 | 6 级 | 默认推理深度 |
| 精简模式 | 关闭/轻量/标准/极致 | Ponytail 懒人模式默认强度 |

### 5.4 高级

- **settings.json 编辑器**：直接编辑 `~/.claude/settings.json`
- **CC 管理快捷入口**：一键跳转管理面板的 8 个标签页

## 6. 管理中心（8 合 1）

导航栏扳手图标打开，8 个标签页切换。

### 6.1 插件

- 列出所有已启用插件，显示描述
- 启用/停用插件
- 添加插件：点击按钮自动发送 `/plugin install` 命令

### 6.2 MCP 服务器

- 扫描多源 MCP 配置：`~/.claude.json`、`.mcp.json`、已启用插件
- 显示服务名、类型（stdio/sse/http）、连接状态
- 启用/禁用 MCP 服务器
- 自动生成中文描述（AI + SQLite 缓存）

### 6.3 Skills

- 按来源分组：自定义（`~/.claude/skills/`）+ 已启用插件
- 显示 SKILL.md 头部 YAML 的描述（AI 翻译为中文）
- 点击行直接执行该 Skill（`/skill-name` 发送到会话）
- 悬停「查看」按钮编辑 SKILL.md
- 底部「重新翻译」清空缓存重新生成描述

### 6.4 Agents

- 来源：自定义（`~/.claude/agents/`）+ 已启用插件
- 🟢 绿点 = 当前会话使用过，⚪ 灰点 = 未使用
- 用过的排前面，未用的排后面
- 描述支持 AI 翻译

### 6.5 Hooks

- 显示 `settings.json` hooks 字段中的配置
- 事件名 + matcher + 命令详情
- 支持 SessionStart、PreToolUse 等生命周期

### 6.6 Memory

- 以树形结构浏览全局和项目的 Memory 目录
- 点击文件查看/编辑内容

### 6.7 权限规则

- 显示当前权限配置
- 管理已批准的权限场景

### 6.8 输出样式

- 列出 `settings.json` 中配置的输出样式

## 7. 命令面板

**Ctrl+K** 全局调出，支持拼音搜索。

### 7.1 命令分组

| 分组 | 示例命令 |
|------|----------|
| 会话 | 新建会话、重命名、删除、导出、继续最近会话 |
| 文件 | 在资源管理器中打开、复制路径 |
| 视图 | 切换侧边栏、文件面板、禅模式、全屏 |
| 模式 | 切换权限模式、思考深度 |
| 管理 | 打开管理面板、设置 |
| 调试 | 显示上下文窗口详情 |

### 7.2 搜索

- 支持中文关键词搜索
- 支持拼音首字母搜索（如 "xh" → "新建会话"）
- 显示最近使用的命令

## 8. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建会话 |
| `Ctrl+B` | 切换侧边栏 |
| `Ctrl+E` | 切换文件面板 |
| `Ctrl+K` | 命令面板 |
| `Ctrl+,` | 打开设置 |
| `Enter` | 发送消息 |
| `Shift+Enter` | 消息换行 |
| `F2` | 重命名当前会话 |

## 9. 附录：数据存储位置

| 数据 | 位置 |
|------|------|
| API 设置 | 浏览器 `localStorage`（`cc-gui-settings`） |
| 会话 + 消息 | SQLite（`src-tauri/` 下的 `cc-gui.db`） |
| 权限场景 | SQLite `approved_scenarios` 表 |
| CC 配置 | `~/.claude/settings.json` |
| 插件缓存 | `~/.claude/plugins/cache/` |
| 项目记忆 | `~/.claude/projects/<slug>/memory/` |
| 主题/语言 | 浏览器 `localStorage`（Pinia settings store） |
